Software Projects

FireDrone Website (2020/2021)

Team project designing hardware module which relays necessary information for creation of IFR to BC Wildfire Services and detects fire using image processing via a user-friendly and comprehensive website. I am currently working on prototyping the UI design of this website with Figma (pictured below), in order to later build the platform using React and NodeJS.

PlantStalk Android App (2020)

Term project integrating a sensor module assessing the soil moisture, temperature and light of garden plants. This information is displayed to users via an Android smartphone app. We developed our app using Android Studio - this included features such as notifications, bluetooth connection and data graphs.

Hardware Engineering Projects

PlantStalk Sensor Module (2020)

Term project integrating a sensor module assessing the soil moisture, temperature and light of garden plants. We used the Arduino Bluno chip to gather data from temperature, light and capacitive soil moisture sensors. We then communicated with our Android app via bluetooth.

Line Following Robot (2019)

Individual project designing and programming a line following robot to compete in a maze competition at the end of term. This robot used photoresistors to detect the difference between the black line and the white paper, in order to follow its path.

CAD Modelling Projects

UBC Rocket Nose Cone Design (2018/2019)

During my time with UBC Rocket, I got to the chance to CAD a C-class rocket nose cone in SolidWorks - we later got to print our designs and assemble our own C-class rockets, which we launched. During my time there, I also contributed to the competition team by simulating rocket flight patterns in RASAero II and OpenRocket.
