# About Me:

Hi! My name is Milena, I'm an Integrated Engineering student at the University of British Columbia. In my studies I focus on Mechanical and Computer engineering,
which allows me to combine my passions for aerostructure engineering and computer programming. 

I am driven by a passion for learning, which is why I have continued growing my skills by joining design teams, such as UBC Rocket and UBC Mars Colony, and working 
on team projects. One such project integrated a sensor module assessing the soil moisture, temperature and light of garden plants while notifying users of extreneours 
conditions via an Android app. I lead the development of the app in Android Studio, teaching myself and the team how to work with Java and use object oriented 
programming concepts.  

I am currently working on another team project which leverages the power of aircraft surveillance to detect forest fires. We are prototyping a hardware module which relays sensor
data and photos to a Raspberry Pi, which then processes the information to be used for the creation of an IFR (Initial Fire Report). I have been working on prototyping the UX/UI
for a web platform to streamline the documentation of IFR's, in order to build this platform using React and NodeJS with the team.

I have worked with C in Visual Studio, as well as in the Arduino IDE, taught myself Python scripting through Codecademy and used Java to develop an Android app. I thrive in 
fast-paced, self-motivated environments and I love collaborating with others to create innovative solutions.
