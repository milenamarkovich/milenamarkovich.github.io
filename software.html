Software Projects

Youtube to MP3 Conversion Python Project (2021)

As someone who enjoys a lot of non-English music, I frequently found myself frustrated by the shortcomings of the Spotify database. I figured the only way I could add these songs to my playlists was by going through the tedious process of converting Youtube videos to MP3 files, and adding them to my Local Files Library. In order to automate this tedious process I wrote a Python script which prompts the user for a Youtube URL and target folder and downloads the MP3 file of the audio to the given folder.

You can find the project Github Repository here!

PlantStalk Android App (2020)

Term project integrating a sensor module assessing the soil moisture, temperature and light of garden plants. This information is displayed to users via an Android smartphone app. We developed our app using Android Studio - this included features such as notifications, bluetooth connection and data graphs.
